## 快速学习顺序建议

1. 先学 `Claude 101`
2. 再看 `AI Fluency: Frameworks & Foundations`
3. 想写代码就接 `Building with the Claude API` 和 `Claude Code in Action`
4. 想做工具生态就接 `Intro to Model Context Protocol` 和 `MCP: Advanced Topics`
5. 最后再看第三方指南，作为实战补充
